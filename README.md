# Lettuce Chat 🥬

**Lettuce Chat** is a modern, lightweight, and secure messaging application built with **Jetpack Compose** and **Firebase**. It focuses on a fluid user experience, real-time communication, and a sleek, minimalistic design.

## ✨ Features

* **Real-time Messaging:** Instant message delivery powered by Firebase.
* **Secure Auth:** User registration and login via Firebase Authentication.
* **Modern UI:** Built entirely with Jetpack Compose for a reactive, smooth feel.
* **Clean Architecture:** Implements **MVVM** pattern for scalable and maintainable code.
* **Visuals:** A curated minimalist aesthetic using custom color palettes.

---

## 🛠 Tech Stack

* **Language:** [Kotlin](https://kotlinlang.org/)
* **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
* **Backend:** [Firebase](https://firebase.google.com/) (Auth & Firestore/Realtime DB)

## 🚀 Getting Started

To get a local copy up and running, follow these steps:

### Prerequisites
* Android Studio Ladybug (or newer)
* A Firebase project setup in the [Firebase Console](https://console.firebase.google.com/)

### Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/yourusername/Lettuce-Chat.git](https://github.com/yourusername/Lettuce-Chat.git)
    ```
2.  **Add Firebase configuration:**
    * Download your `google-services.json` from Firebase.
    * Place it in the `app/` folder of the project.
3.  **Build & Run:**
    * Sync Gradle and run the app on your emulator or physical device.
